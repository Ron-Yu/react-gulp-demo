# react-gulp-demo

## Introduction
It's the demo to start react js application.

## Build tools
- [gulp](http://gulpjs.com/)
- [browserify](http://browserify.org/)
- [browser-sync](http://www.browsersync.io/)

## Steps
1. install [node.js](https://nodejs.org/en/)
2. npm install gulp -g
3. npm install
4. gulp
